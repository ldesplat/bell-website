Bell Examples for different OAuth providers

This is only meant as an example for the hapijs/bell project. If you try to log in with the different providers, none of your information is saved or transmitted to third parties. All of your information is stored in cookie or the browser's local storage and expires once your close your browser. You can audit the full source code of the currently running website at https://bell.now.sh/_src

If you have any issues/comments or want to contribute, the project is located at https://github.com/ldesplat/bell-website
